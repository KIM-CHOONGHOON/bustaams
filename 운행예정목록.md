# 운행 예정 목록 (UpcomingTrips)

기사 **운행 예정 목록** 화면은 `UpcomingTripsModal`로 제공되며, 조회 API·비즈니스 규칙의 상세는 [`운행예정목록 화면.md`](./운행예정목록%20화면.md)를 참고한다.

---

## 1. 포함하지 않는 UI (삭제됨)

| 항목 | 설명 |
|------|------|
| **새 버스 등록하기** | 상단 우측 액션 버튼 제거 |
| **지도 배경 섹션** | 지도 이미지·차량 위치 문구 등 지도 블록 전체 제거 |
| **「진행 중」 탭** | 탭 메뉴에서 제거 (`전체 일정` · `완료된 운행`만 유지) |
| **전국 배차 현황** | 지역별 건수·막대 그래프 섹션 제거 |

원본 HTML에 있던 **좌측 사이드 메뉴·상단 TopAppBar**는 모달에서 처음부터 포함하지 않는다.

---

## 2. 유지되는 주요 UI

| 영역 | 내용 |
|------|------|
| **헤더** | 「운행 요약」·제목 **「운행 예정 목록」**·상태/안내 문구 |
| **탭** | **전체 일정**(활성 스타일) · **완료된 운행** (클릭 동작은 추후 연동 가능) |
| **본문 좌측** | 우선 카드(첫 일정)·보조 카드 그리드·빈 상태 문구 |
| **우측** | **주간 요약** 카드(예상 주간 수익·총 운행 거리·목표 달성률 등) |
| **하단** | 푸터 링크(개인정보 처리방침·이용 약관·쿠키 설정·©) |
| **모달** | 흰색 패널·**우상단 X** 닫기·배경 오버레이·`Esc` 닫기 |
