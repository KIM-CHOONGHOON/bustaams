# 총 운임 비교 섹션 (DashboardTopSection — 운임 카드)

`DriverDashboard` 상단 그리드 좌측 카드(구 「총 수익」)를 **당월·전월 운임 합계 비교**로 표시하기 위한 UI·데이터·API 정의입니다.

---

## 1. 화면 요구사항

| 항목 | 규칙 |
|------|------|
| **제목** | 시스템 일자 기준 **`YYYY년 MM월 총 운임 금액`** (예: `2026년 4월 총 운임 금액`) |
| **주요 금액** | **당월 합계액** — 아래 [2절](#2-당월합계) |
| **하단 비교 문구** | **전월 대비 차액** — 아래 [3절](#3-전월-대비-색상) |
| **데이터 출처** | `TB_BUS_RESERVATION` · `RES_STAT = 'DONE'` · `DRIVER_BIDDING_PRICE` 합계 |
| **일자 기준** | 행이 완료 처리된 시점으로 **`MOD_DT`** 를 사용 (같은 달 집계 구간에 포함되는지 판별) |

---

## 2. 당월·전월 합계

### 2-1. 전월 합계

- **구간:** 시스템 현재 기준 **직전 달 1일 00:00 ~ 직전 달 말일** (해당 월에 속하는 `MOD_DT`)
- **조건:** `RES_STAT = 'DONE'`, `DRIVER_UUID` = 로그인 기사
- **집계:** `SUM(DRIVER_BIDDING_PRICE)`

SQL 조건 예:

```sql
MOD_DT >= DATE_SUB(DATE_FORMAT(CURDATE(), '%Y-%m-01'), INTERVAL 1 MONTH)
AND MOD_DT < DATE_FORMAT(CURDATE(), '%Y-%m-01')
```

### 2-2. 당월 합계

- **구간:** **현재 년·월 1일**부터 **오늘(`CURDATE`)** 까지 (`DATE(MOD_DT) <= CURDATE()`)
- **조건:** 동일 (`DONE`, 해당 기사)
- **집계:** `SUM(DRIVER_BIDDING_PRICE)`

```sql
MOD_DT >= DATE_FORMAT(CURDATE(), '%Y-%m-01')
AND DATE(MOD_DT) <= CURDATE()
```

---

## 3. 전월 대비 색상

`diffFromPrevious = currentMonthTotal - previousMonthTotal`

| 조건 | 색상 | 표시 예 |
|------|------|---------|
| 당월 ≥ 전월 (`diffFromPrevious >= 0`) | **붉은색** | `전월 대비 +₩…` |
| 당월 &lt; 전월 (`diffFromPrevious < 0`) | **파란색** | `전월 대비 -₩…` |

아이콘: 차액 ≥ 0 이면 `trending_up`, &lt; 0 이면 `trending_down`.

## 4. REST API

### `GET /api/driver/dashboard?uuid={기사UUID}`

**응답 필드 (총 운임 관련)**

| 필드 | 타입 | 설명 |
|------|------|------|
| `year` | number | 제목용 연도 |
| `month` | number | 제목용 월 (1–12) |
| `currentMonthTotal` | number | 당월 합계 |
| `previousMonthTotal` | number | 전월 합계 |
| `diffFromPrevious` | number | 당월 − 전월 |
| `compareTone` | `'gte_prev' \| 'lt_prev'` | `gte_prev` → 붉은색, `lt_prev` → 파란색 |

**부가 (활성 입찰 카드):** `bidCount`, `bidAmountSum`, `confirmCount`, `confirmAmountSum`, `activeBids`(=`bidCount`) — 상세는 [`활성 입찰 섹션.md`](./활성%20입찰%20섹션.md).
