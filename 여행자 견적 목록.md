# 여행자 견적 목록 (ListOfTravelerQuotations) 화면 명세

본 문서는 `busTaams` 프로젝트의 **여행자 견적 목록** 모달 화면에 대한 UI·데이터·API·설계 기준을 정의합니다.  
개발 환경 기준은 [`ARCHITECTURE.md`](./ARCHITECTURE.md)를 따릅니다.

---

## 1. 화면 개요

| 항목 | 내용 |
|------|------|
| **화면명** | 여행자 견적 목록 |
| **영문 식별자** | `ListOfTravelerQuotations` |
| **접근** | 운전기사 대시보드 → 바로가기 메뉴 → **「여행자 견적 목록」** |
| **형태** | 오버레이 모달 (`z-[150]`) |
| **목적** | 여행자가 버스 기사 다수에게 등록한 **역경매(Reverse Auction)** 견적 요청 목록을 기사에게 제공, 기사가 원하는 견적을 선택·입찰할 수 있도록 함 |

### 1-1. 역경매(역경매) 개념

```
여행자 (1명)
  └─ 버스 대절 견적 요청 등록 (TB_BID_REQUEST)
        ↓
기사 다수 (N명) 가 목록을 조회하고 견적을 제출
  └─ 기사가 「여행자 견적 목록」에서 STATUS = 'OPEN' 인 항목 확인
  └─ 항목 클릭 → 「TravelerQuoteRequestDetails」 모달 호출 → 상세 조회 및 입찰
```

## 5. 데이터 모델 (DB 테이블)

### 5-1. TB_BID_REQUEST — 소비자 입찰 요청 메인 (조회 대상)

| 컬럼 | 타입 | 목록 표시 |
|------|------|---------|
| `REQUEST_ID` | `BIGINT` PK (Auto Increment) | 카드 클릭 시 `TravelerQuoteRequestDetails`로 전달 (`requestId`) |
| `USER_ID` | `VARCHAR(300)` | — |
| `BUS_TYPE` | `VARCHAR(50)` | **희망 버스 종류** |
| `PASSENGER_CNT` | `INT` | **탑승 인원** |
| `CALC_BUS_CNT` | `INT` | — |
| `BUS_CNT` | `INT` | **필요 차량 대수** |
| `START_DT` | `DATETIME` | **출발 일시** |
| `END_DT` | `DATETIME` | **도착 일시** |
| `ROUND_TRIP_YN` | `CHAR(1)` | **왕복 여부** ('Y' 이면 「왕복」 배지 표시) |
| `START_ADDR` | `TEXT` | **출발지 장소명** |
| `END_ADDR` | `TEXT` | **도착지 장소명** |
| `EST_TOTAL_SERVICE_PRICE` | `DECIMAL(15,2)` | **개산 서비스 총액** |
| `COMMENT` | `TEXT` | **세부 요구사항** |
| `STATUS` | `VARCHAR(20)` | 조회 조건: `'OPEN'` 만 추출 |
| `REG_DT` | `DATETIME` | **등록 일시** |

### 5-2. TB_BID_WAYPOINT — 입찰 요청 경유지 상세 (건수만 집계)

| 컬럼 | 역할 |
|------|------|
| `WAYPOINT_ID` | PK |
| `REQUEST_ID` | `TB_BID_REQUEST.REQUEST_ID` 와 JOIN (COUNT 집계 → `waypointCount`) |
| `SORT_ORDER` | 방문 순서 |
| `WAYPOINT_ADDR` | 경유지 장소명 |

## 8. REST API

### 8-1. GET /api/list-of-traveler-quotations

**역경매 견적 요청 목록 조회 — STATUS = 'OPEN'**

| 항목 | 내용 |
|------|------|
| **메서드** | `GET` |
| **경로** | `/api/list-of-traveler-quotations` |
| **응답** | JSON — `total` + `items` 배열 |
| **데이터 없을 경우** | 개발용 더미 데이터 반환 |

**SQL 쿼리 (핵심)**

```sql
SELECT
    r.REQUEST_ID              AS requestId,
    r.USER_ID                 AS userId,
    r.BUS_TYPE                AS busType,
    r.BUS_CNT                 AS busCnt,
    r.PASSENGER_CNT           AS passengerCnt,
    r.START_DT                AS startDt,
    r.END_DT                  AS endDt,
    r.START_ADDR              AS startAddr,
    r.END_ADDR                AS endAddr,
    r.ROUND_TRIP_YN           AS roundTripYn,
    r.EST_TOTAL_SERVICE_PRICE AS estTotalServicePrice,
    r.COMMENT                 AS comment,
    r.STATUS                  AS status,
    r.REG_DT                  AS regDt,
    COUNT(w.WAYPOINT_ID)      AS waypointCount
FROM TB_BID_REQUEST r
LEFT JOIN TB_BID_WAYPOINT w ON w.REQUEST_ID = r.REQUEST_ID
WHERE r.STATUS = 'OPEN'
GROUP BY r.REQUEST_ID
ORDER BY r.REG_DT DESC;
```

**응답 스키마**

```json
{
  "total": "number",
  "items": [
    {
      "requestId":            "number",
      "userId":               "string",
      "busType":              "string",
      "busCnt":               "number",
      "passengerCnt":         "number",
      "startDt":              "datetime",
      "endDt":                "datetime",
      "startAddr":            "string",
      "endAddr":              "string",
      "roundTripYn":          "Y | N",
      "estTotalServicePrice": "number",
      "comment":              "string | null",
      "status":               "OPEN",
      "regDt":                "datetime",
      "waypointCount":        "number"
    }
  ]
}
```

## 9. 프론트엔드 연동 흐름

```
App.jsx
  └─ showQuotationModal (useState)
       ├─ DriverDashboard → QuickMenu → '여행자 견적 목록' 클릭
       │     → setShowQuotationModal(true)
       └─ {showQuotationModal &&
             <ListOfTravelerQuotations close={...} currentUser={user} />}
               │
               ├─ useEffect on mount
               │     → fetch GET /api/list-of-traveler-quotations
               │     → setItems(result.items)
               │
               └─ 카드 클릭 → setSelectedRequestId(item.requestId)
                     └─ {selectedRequestId &&
                           <TravelerQuoteRequestDetails
                             requestId={selectedRequestId}
                             close={() => setSelectedRequestId(null)}
                             currentUser={currentUser}
                           />}
```
